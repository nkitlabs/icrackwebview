### Noted

Steps to initiate react project
1. initate package.json

    ```npm init -y```

2. install react package

    `npm install react react-dom --save`

3. install webpack package

    `npm install webpack webpack-dev-server webpack-cli --save`

4. install barbell

    `npm install babel-core babel-loader @babel/preset-env @babel/preset-react html-webpack-plugin --save-dev`

5. set webpack.config.js
6. set script command in package.json
7. set .barbelrc
8. create index.html / App.js / main.js
9. run `npm start`
